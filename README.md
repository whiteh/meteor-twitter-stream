#TwitterStream

Requires npm container
`meteor add meteorhacks:npm`

Add `private/twitter.json` with the relevant twitter credentials.
```{
     "consumer": {
         "key": "xxx",
         "secret": "xxx"
     },
     "access_token": {
         "key": "xxx",
         "secret": "xxx"
     },
     "hashtag": "#javascript"
 }```
